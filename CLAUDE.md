# BMC PROJECT RULES

## Design System

**Typography:** Headings (H1-H6) — 'Mont' (Bold/Black). Body/UI — 'Inter' (Regular/Medium).

**Spacing:** 8px Grid System (p-2, p-4, gap-8).

**Colors:** Primary Blue — #2563EB.

## Technical

**Framework:** React + Vite + Tailwind v4.

**Icons:** Lucide React.

**Responsiveness:** Mobile-first approach.

## Workflow

Always use /typeset for text and /arrange for layout.

Keep components clean and modular.
